This project started as a handwritten-list of songs I intend to screen for radio play.

The realities of music media quickly made me realize a spreadsheet wouldn't work because of the variations in crediting and so forth. 

My first database was born. Baby's first SQL project.

In the end, one should be able to recreate my database by running data.sql with the schema set, and query music by song, artist, genre, language, country, in their original script/alphabet or romanized.

INSTRUCTIONS:

Want to build your own database? 
-run add_song.py; add songs one by one

Want to farm my songs?
-use data.sql and schema.sql to build my database on your own.
